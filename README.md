# ⚽ Futbolcu Tahmin Etme

Futbolcu fotoğraflarına bakarak hangi futbolcuyu tahmin ettiğini keşfedebileceğin eğlenceli bir mobil quiz uygulaması. React Native ve Expo ile geliştirilmiştir.

---

## 🚀 Özellikler

- 🔍 Futbolcu fotoğraflarından tahmin yapma
- 📱 iOS ve Android desteği
- 🎨 NativeWind (Tailwind CSS) ile modern ve şık arayüz
- 🗂️ Zustand ile sade ve hızlı state yönetimi
- 🔀 Expo Router ile dosya tabanlı navigasyon

## 📦 Kurulum

### Gereksinimler

- Node.js (v18+)
- npm veya yarn
- Expo CLI
- iOS için: Xcode (macOS)
- Android için: Android Studio

### Adımlar

```bash
# Repoyu klonla
git clone https://github.com/cemgunal12/FutbolcuTahminEtme.git
cd FutbolcuTahminEtme

# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npm start
```

---

## ▶️ Çalıştırma

```bash
# Expo geliştirici araçlarını aç
npm start

# Android emülatörde çalıştır
npm run android

# iOS simülatörde çalıştır
npm run ios

# Web'de çalıştır
npm run web
```
